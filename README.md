## Задание 2
`Установите и запустите memcached. Приведите скриншот systemctl status memcached, где будет видно, что memcached запущен.`

### Решение
* `Устанавливаю Memcached`
```
sudo apt install memcached
```
* `Проверяю статус`
```
sudo systemctl status memcached
```
